# Core 模块

本模块包含 ModelViewer 的核心数据结构，定义了应用程序中使用的基础数据类型。

## 文件说明

### CpuMesh.h
定义了 CPU 端的网格数据结构。

**主要结构：**
```cpp
struct CpuMesh {
    struct Vertex {
        QVector3D position;    // 顶点位置
        QVector3D normal;      // 顶点法线
        QVector2D texCoord;    // 纹理坐标
    };
    
    QVector<Vertex> vertices;  // 顶点数组
    QVector<quint32> indices;  // 索引数组
    BoundingBox bbox;         // 包围盒
};
```

**功能特点：**
- 支持位置、法线、纹理坐标的顶点格式
- 使用索引缓冲区优化渲染性能
- 内置包围盒用于快速碰撞检测和视锥剔除

### Light.h
定义场景中的光照参数。

**主要功能：**
- 光源位置、颜色、强度等属性
- 支持点光源、方向光、聚光灯等类型
- 为 PBR 渲染提供光照参数

### Material.h
定义材质属性和参数。

**主要功能：**
- 漫反射、镜面反射、自发光等基础材质属性
- PBR 材质参数（金属度、粗糙度等）
- 纹理绑定和采样参数

## 设计原则

- **数据导向**：专注于数据结构定义，不包含业务逻辑
- **内存效率**：使用 Qt 容器类，优化内存布局
- **类型安全**：使用强类型定义，减少类型转换错误
- **可扩展性**：结构设计便于添加新的顶点属性和材质参数

## 使用说明

这些核心数据结构被其他模块广泛使用：
- `loader` 模块将加载的模型数据填充到 `CpuMesh` 结构中
- `renderer` 模块将这些数据上传到 GPU 进行渲染
- `ui` 模块使用这些结构进行参数编辑和显示

## 注意事项

- 所有结构都使用 Qt 的数据类型（QVector3D、QVector2D 等）
- 顶点数据采用右手坐标系
- 法线向量假设为单位向量
