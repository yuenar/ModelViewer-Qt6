# ModelViewer-Qt6 RHI 新增功能说明

## 概述

基于Qt5架构设计文档，已将以下功能从Qt5 OpenGL版本迁移到Qt6 RHI版本，并扩展了12种渲染模式。

---

## 新增渲染模式

### 1. **Phong 光照模式** (模式 0)
- 基础的Phong/Blinn-Phong光照模型
- 支持环境光、漫反射、高光计算
- 可调节材质参数（环境光、漫反射、高光、光泽度）

### 2. **线框模式** (模式 1)
- 显示模型的边框线条
- 用于检查模型拓扑结构

### 3. **顶点法线可视化** (模式 2)
- 从每个顶点绘制法线向量
- 帮助调试法线方向

### 4. **面法线可视化** (模式 3)
- 从每个三角形面心绘制法线向量
- 用于检查面的法线方向

### 5. **顶点法线增强可视化** (模式 4)
- 增强的顶点法线显示
- 更清晰的法线可视化

### 6. **平面着色模式** (模式 5)
- 使用平面着色而非平滑着色
- 显示模型的几何结构

### 7. **PBR 渲染模式** (模式 6) ✓ 已实现
- 基于物理的渲染（Physically Based Rendering）
- 使用 BRDF 着色器计算
- 支持金属度和粗糙度参数

### 8. **阴影映射模式** (模式 7) ✓ 已实现
- 实时阴影计算
- 使用深度贴图进行阴影映射
- 支持软阴影效果

### 9. **天空盒模式** (模式 8) ✓ 已实现
- 环境映射和天空盒渲染
- 支持立方体贴图
- 增强场景的视觉效果

### 10. **选择模式** (模式 9) ✓ 已实现
- 对象选择和高亮显示
- 支持拾取和交互
- 用于模型编辑和检查

### 11. **裁剪平面模式** (模式 10) ✓ 已实现
- 使用裁剪平面切割模型
- 查看模型内部结构
- 支持动态调整裁剪平面

### 12. **分屏显示模式** (模式 11) ✓ 已实现
- 同时显示多个渲染视图
- 对比不同渲染效果
- 支持自定义分屏布局

## 核心功能实现

### 背景渐变
- 支持自定义顶部和底部背景颜色
- 使用专用的背景渲染管线
- 在主场景之前渲染

### 材质系统
- 环境光颜色 (Ambient)
- 漫反射颜色 (Diffuse)
- 高光颜色 (Specular)
- 光泽度 (Shininess)
- 金属度 (Metallic) - 用于 PBR 模式
- 粗糙度 (Roughness) - 用于 PBR 模式

### 光源系统
- 光源位置
- 光源颜色
- 可实时调节
- 支持阴影计算

### 相机控制
- 左键拖拽：轨迹球旋转
- 中键拖拽：平移
- 滚轮：缩放
- 透视投影 / 正交投影切换

## 着色器支持

已编译并集成的着色器：

### 基础着色器（模式 0-5）
- `phong.vert / phong.frag` - Phong光照
- `wireframe.vert / wireframe.frag` - 线框
- `normals.vert / normals.frag` - 顶点法线
- `face_normal.vert / face_normal.frag` - 面法线
- `vertex_normal.vert / vertex_normal.frag` - 顶点法线可视化
- `background.vert / background.frag` - 背景渐变

### 扩展着色器（模式 6-11）
- `brdf.vert / brdf.frag` - PBR 渲染
- `shadow_mapping_depth.vert / shadow_mapping_depth.frag` - 阴影映射
- `skybox.vert / skybox.frag` - 天空盒
- `selection.vert / selection.frag` - 选择模式
- `clipping_plane.vert / clipping_plane.frag` - 裁剪平面
- `splitScreen.vert / splitScreen.frag` - 分屏显示

### 其他着色器
- `axis.vert / axis.frag` - 坐标轴显示
- `clipped_mesh.vert / clipped_mesh.frag` - 裁剪网格
- `debug_quad.vert / debug_quad_depth.frag` - 调试四边形
- `light_cube.vert / light_cube.frag` - 光源立方体
- `irradiance_convolution.frag` - 辐照度卷积
- `text.vert / text.frag` - 文本渲染

## 使用示例

### 切换渲染模式
```cpp
// 通过菜单或代码
m_rhiWidget->setRenderMode(2);   // 顶点法线模式
m_rhiWidget->setRenderMode(6);   // PBR 模式
m_rhiWidget->setRenderMode(8);   // 天空盒模式
```

### 设置背景颜色
```cpp
QVector3D topColor(0.2f, 0.3f, 0.5f);
QVector3D botColor(0.1f, 0.1f, 0.2f);
m_rhiWidget->setBackgroundColors(topColor, botColor);
```

### 调节材质
```cpp
Material mat;
mat.ambient = QVector3D(0.2f, 0.2f, 0.2f);
mat.diffuse = QVector3D(0.8f, 0.8f, 0.8f);
mat.specular = QVector3D(1.0f, 1.0f, 1.0f);
mat.shininess = 32.0f;
m_rhiWidget->setMaterial(mat);
```

## 已知限制

1. 几何着色器不支持（HLSL 兼容性问题）
2. 法线可视化的长度固定为 0.1f（可扩展为可配置）
3. 背景颜色设置目前为简单的两色渐变
4. 阴影映射需要额外的深度纹理支持
5. 天空盒需要立方体贴图资源
