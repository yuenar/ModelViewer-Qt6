# ModelViewer-Qt6 RHI 新增功能说明

## 概述

基于Qt5架构设计文档，已将以下功能从Qt5 OpenGL版本迁移到Qt6 RHI版本：

---

## 新增渲染模式

### 1. **Phong 光照模式** (模式 0)
- 基础的Phong/Blinn-Phong光照模型
- 支持环境光、漫反射、高光计算
- 可调节材质参数（环境光、漫反射、高光、光泽度）

### 2. **线框模式** (模式 1)
- 显示模型的边框线条
- 用于检查模型拓扑结构

### 3. **顶点法线可视化** (模式 2)
- 从每个顶点绘制法线向量
- 帮助调试法线方向

### 4. **面法线可视化** (模式 3)
- 从每个三角形面心绘制法线向量
- 用于检查面的法线方向

### 5. **顶点法线可视化** (模式 4)
- 增强的顶点法线显示
- 更清晰的法线可视化

### 6. **平面着色模式** (模式 5)
- 使用平面着色而非平滑着色
- 显示模型的几何结构

## UI 菜单增强

### View 菜单
- **Fit to View** - 自动适配视角到模型
- **Wireframe** - 切换线框模式
- **Vertex Normals** - 显示顶点法线
- **Face Normals** - 显示面法线
- **Vertex Normals Visualization** - 增强法线可视化
- **Flat Shading** - 平面着色模式
- **Orthographic Projection** - 正交投影切换

### Tools 菜单
- **Material Settings** - 材质参数调节
- **Light Settings** - 光源参数调节
- **Background Settings** - 背景颜色设置
- **Screenshot** - 截图导出

---

## 核心功能实现

### 背景渐变
- 支持自定义顶部和底部背景颜色
- 使用专用的背景渲染管线
- 在主场景之前渲染

### 材质系统
- 环境光颜色 (Ambient)
- 漫反射颜色 (Diffuse)
- 高光颜色 (Specular)
- 光泽度 (Shininess)

### 光源系统
- 光源位置
- 光源颜色
- 可实时调节

### 相机控制
- 左键拖拽：轨迹球旋转
- 中键拖拽：平移
- 滚轮：缩放
- 透视投影 / 正交投影切换

## 着色器支持

已编译并集成的着色器：
- `phong.vert / phong.frag` - Phong光照
- `wireframe.vert / wireframe.frag` - 线框
- `normals.vert / normals.frag` - 顶点法线
- `face_normal.vert / face_normal.frag` - 面法线
- `vertex_normal.vert / vertex_normal.frag` - 顶点法线可视化
- `background.vert / background.frag` - 背景渐变

---

## 使用示例

### 切换渲染模式
```cpp
// 通过菜单或代码
m_rhiWidget->setRenderMode(2);  // 顶点法线模式
```

### 设置背景颜色
```cpp
QVector3D topColor(0.2f, 0.3f, 0.5f);
QVector3D botColor(0.1f, 0.1f, 0.2f);
m_rhiWidget->setBackgroundColors(topColor, botColor);
```

### 调节材质
```cpp
Material mat;
mat.ambient = QVector3D(0.2f, 0.2f, 0.2f);
mat.diffuse = QVector3D(0.8f, 0.8f, 0.8f);
mat.specular = QVector3D(1.0f, 1.0f, 1.0f);
mat.shininess = 32.0f;
m_rhiWidget->setMaterial(mat);
```

## 已知限制

1. 几何着色器支持依赖于RHI后端（某些平台可能不支持）
2. 法线可视化的长度固定为0.1f（可扩展为可配置）
3. 背景颜色设置目前为简单的两色渐变
