# ModelViewer-Qt6 RHI 新增功能说明

## 概述

基于 Qt6 RHI 架构，ModelViewer-Qt6 实现了完整的 3D 模型查看器功能。本文档详细说明已实现的功能、技术架构和后续扩展计划。

---

## 已实现的核心功能

### 1. 多格式模型加载
- **支持格式**：OBJ、FBX、GLTF、3DS、DAE、STL 等（通过 Assimp）
- **加载方式**：异步加载（可扩展）
- **数据管理**：CPU 端 CpuMesh 和 GPU 端 RhiMesh 的双层管理
- **包围盒计算**：自动计算模型包围盒用于视图适配

### 2. 12 种渲染模式

#### 基础渲染模式（模式 0-5）
| 模式 | 名称 | 着色器 | 状态 | 说明 |
|------|------|--------|------|------|
| 0 | Phong 光照 | phong.vert/frag | ✓ 完成 | 经典 Phong 光照模型 |
| 1 | 线框显示 | wireframe.vert/frag | ✓ 完成 | 显示模型网格拓扑 |
| 2 | 顶点法线 | normals.vert/frag | ✓ 完成 | 从顶点绘制法线向量 |
| 3 | 面法线 | face_normal.vert/frag | ✓ 完成 | 从面心绘制法线向量 |
| 4 | 法线增强 | vertex_normal.vert/frag | ✓ 完成 | 增强的法线可视化 |
| 5 | 平面着色 | phong.vert/frag | ✓ 完成 | 无光照的平面颜色 |

#### 高级渲染模式（模式 6-11）

```
                          ↓
┌─────────────────────────────────────────────────────┐
│                   RhiMesh 管理                       │

                          ↓

```

### 数据结构

#### FrameUBOData（帧数据）
```cpp
struct FrameUBOData {
    mat4 model;           // 模型矩阵
    mat4 view;            // 视图矩阵
    mat4 projection;      // 投影矩阵
    mat4 normalMatrix;    // 法线矩阵
    vec3 lightPos;        // 光源位置
    vec3 lightColor;      // 光源颜色
    vec3 viewPos;         // 观察者位置
};
```

#### MaterialUBOData（材质数据）
```cpp
struct MaterialUBOData {
    vec3 ambient;         // 环境光
    float shininess;      // 光泽度
    vec3 diffuse;         // 漫反射
    vec3 specular;        // 高光
};
```

#### BackgroundUBOData（背景数据）
```cpp
struct BackgroundUBOData {
    vec4 topColor;        // 顶部颜色
    vec4 botColor;        // 底部颜色
};
```

### 着色器系统

#### 编译流程
```
.vert/.frag → qsb 编译器 → .qsb 二进制 → 运行时加载
```

#### 支持的着色器
- **顶点着色器**：18 个 .vert 文件
- **片段着色器**：18 个 .frag 文件
- **总计**：36 个着色器文件

#### 着色器特性
- ✓ SPIR-V 编译支持
- ✓ 多平台兼容（GLSL、HLSL、MSL）
- ✓ 统一的 UBO 布局（std140）
- ✓ 位置限定符（layout location）

## 已实现的功能详解

### 1. 12 种渲染模式的实现

#### 基础模式（0-5）
- **Phong 模式**：标准的 Phong 光照计算
- **线框模式**：使用 GL_LINE 拓扑绘制
- **法线模式**：从顶点绘制法线向量
- **面法线模式**：从三角形面心绘制法线
- **法线增强**：更清晰的法线可视化
- **平面着色**：禁用光照计算

#### 高级模式（6-11）
- **PBR 模式**：使用 BRDF 着色器计算
- **阴影模式**：深度贴图阴影映射
- **天空盒模式**：立方体贴图环境
- **选择模式**：高亮选中物体
- **裁剪模式**：使用裁剪平面切割
- **分屏模式**：同时显示多个视图

### 2. 菜单系统

#### 互斥性设计
- 所有渲染模式互斥（同时只能一个激活）
- 投影模式独立切换
- 背景设置独立操作

#### 快捷键支持
- F：聚焦视图
- 空格：重置视图（可扩展）

### 3. 材质和光照对话框

#### MaterialDialog
- 环境光颜色选择
- 漫反射颜色选择
- 高光颜色选择
- 光泽度滑块调节

#### LightDialog
- 光源位置 XYZ 输入
- 光源颜色选择
- 实时预览

### 4. 背景颜色设置

#### 实现方式
- 两个 QColorDialog 分别选择顶部和底部颜色
- 实时更新背景渐变
- 支持自定义颜色

### 5. 快捷键系统

#### 实现方式
- 在 RhiWidget::keyPressEvent() 中处理所有快捷键
- 支持 8 个主要快捷键和 7 个视图预设快捷键
- 自动重复事件过滤，避免连续触发

#### 快捷键列表
| 快捷键 | 功能 | 实现方式 |
|--------|------|---------|
| F | 聚焦视图 | 调用 fitToView() |
| Space | 重置视图 | 调用 m_camera.reset() |
| W | 切换线框 | 在模式 0 和 1 之间切换 |
| N | 显示法线 | 在模式 0 和 2 之间切换 |
| L | 切换光照 | 在模式 0 和 1 之间切换 |
| S | 切换阴影 | 在模式 0 和 7 之间切换 |
| P | 切换投影 | 调用 toggleProjection() |
| F11 | 全屏 | 调用 window()->showFullScreen() |
| 1-7 | 视图预设 | 调用对应的视图预设函数 |
| F12 | 截图 | 触发截图菜单项 |

### 6. 视图预设系统

#### 实现方式
- 在 MainWindow 中添加 7 个视图预设槽函数
- 每个预设设置摄像机位置、目标点和上向量
- 支持快捷键 1-7 快速切换

#### 视图预设详情
| 预设 | 快捷键 | 摄像机位置 | 目标点 | 上向量 |
|------|--------|-----------|--------|--------|
| 前视图 | 1 | (0, 0, 10) | (0, 0, 0) | (0, 1, 0) |
| 后视图 | 2 | (0, 0, -10) | (0, 0, 0) | (0, 1, 0) |
| 左视图 | 3 | (-10, 0, 0) | (0, 0, 0) | (0, 1, 0) |
| 右视图 | 4 | (10, 0, 0) | (0, 0, 0) | (0, 1, 0) |
| 顶视图 | 5 | (0, 10, 0) | (0, 0, 0) | (0, 0, -1) |
| 底视图 | 6 | (0, -10, 0) | (0, 0, 0) | (0, 0, 1) |
| 等轴测 | 7 | (10, 10, 10) | (0, 0, 0) | (0, 1, 0) |

### 7. 截图功能增强

#### 实现方式
- 新增 saveScreenshotEx() 方法支持多种格式
- JPEG 格式支持质量选择（0-100）

#### 支持的格式

- BMP（无压缩）

### 8. 模型信息显示

- 使用 QMessageBox 显示统计信息

#### 显示的信息
- 网格数量
- 顶点总数
- 三角形总数
- 包围盒最小值、最大值、尺寸

## 使用示例

### 切换渲染模式
```cpp
// 通过菜单或代码
m_rhiWidget->setRenderMode(0);   // Phong 模式
m_rhiWidget->setRenderMode(6);   // PBR 模式
m_rhiWidget->setRenderMode(8);   // 天空盒模式
```

### 设置背景颜色
```cpp
QVector3D topColor(0.2f, 0.3f, 0.5f);
QVector3D botColor(0.1f, 0.1f, 0.2f);
m_rhiWidget->setBackgroundColors(topColor, botColor);
```

### 调节材质
```cpp
Material mat;
mat.ambient = QVector3D(0.2f, 0.2f, 0.2f);
mat.diffuse = QVector3D(0.8f, 0.8f, 0.8f);
mat.specular = QVector3D(1.0f, 1.0f, 1.0f);
mat.shininess = 32.0f;
m_rhiWidget->setMaterial(mat);
```

### 调节光源
```cpp
Light light;
light.position = QVector3D(10.0f, 10.0f, 10.0f);
light.color = QVector3D(1.0f, 1.0f, 1.0f);
m_rhiWidget->setLight(light);
```

## 已知限制

1. **几何着色器不支持**
   - 原因：HLSL 兼容性问题
   - 影响：法线可视化使用顶点着色器实现

2. **法线长度固定**
   - 当前值：0.1f
   - 改进：可扩展为可配置参数

3. **背景渐变简单**
   - 当前：两色线性渐变
   - 改进：支持多色、径向渐变等

4. **阴影映射基础**
   - 当前：简单的深度贴图
   - 改进：支持软阴影、级联阴影等

5. **性能未优化**
   - 当前：无 LOD、无视锥剔除
   - 改进：实现完整的优化管线

## 编译和运行

### 编译要求
- Qt 6.9.3+
- MSVC 2022 或 GCC 8+
- Vulkan SDK 1.4.335+
- Assimp 5.0+

### 编译命令
```bash
# Windows
cmake -B build -S . -DCMAKE_TOOLCHAIN_FILE=D:/vcpkg/scripts/buildsystems/vcpkg.cmake
cmake --build build

# Linux/macOS
mkdir build && cd build
cmake ..
make -j$(nproc)
```

### 运行
```bash
./build/ModelViewer
```

### 渲染性能
- **帧率**：60+ FPS（1080p，中等模型）
